# Malaga

Multi-purpose project with document/image analysis subtasks.

## Subtasks

| Subtask | Purpose | Input |
|---------|---------|-------|
| `ventaCasaBSA` | Venta de casa BSA | Images in `inputImages/` |
| `ventaDepartamento` | Venta de departamento | Images in `inputImages/` |
| `Pets` | Mascotas | Images in `inputImages/` |
| `Visa` | Visa | Images in `inputImages/` |

## Usage

Place source images in the `inputImages/` folder of each subtask for processing.
